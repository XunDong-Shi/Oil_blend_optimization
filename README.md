# Oil_blend_optimization
## 1.项目内容
炼油厂具有催化裂化、重整、加氢精制等多套生产装置，不同装置产生的组分油的性质、成本各不相同，选择优化成品油调和方案进行调和是提高炼油厂经济效益的重要途径。本项目结合炼油厂实例，分析成品油调和过程建模所需的元素和方法，求解固定成品油产量下的经济指标最优时各调和组分质量提出解决方案，并输出可视化图表。

## 3.实例数据简介
